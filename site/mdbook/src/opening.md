# Opening

This 'book' is a notebook for <https://github.com/20kdc/vrc-stuff>.

It contains documentation on:

* Packages supplied as part of vrc-stuff
* Udon in general
* My personal development practices

It's compiled using both [mdBook](https://github.com/rust-lang/mdBook) and a custom generator for the extern reference.
